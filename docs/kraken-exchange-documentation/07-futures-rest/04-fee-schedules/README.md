# Fee Schedules

Fee schedule endpoints -- platform fee tiers and per-user trading volume lookups.

## Contents

   - `GET /feeschedules`
2. [Get User Fee Schedule Volumes](02_Get-User-Fee-Schedule-Volumes.md) -- Return trailing trading volumes used to determine the user's fee tier.
