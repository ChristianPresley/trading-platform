# Websocket V1

WebSocket API v1 (legacy) -- real-time market data, user data, and trading channels.

## Contents

1. [Admin](01-admin/) -- Connection management, heartbeat, ping/pong, system status, and subscription lifecycle.
2. [Market Data](02-market-data/) -- Public channels for order book, OHLC, spread, ticker, and trade data.
3. [User Data](03-user-data/) -- Authenticated channels for open orders and own trades streaming.
4. [User Trading](04-user-trading/) -- Authenticated methods for placing, amending, cancelling, and editing orders.
