# Users

Embed API endpoints for creating, retrieving, and updating user accounts.

## Contents

1. [Create Embed User](01_Create-Embed-User.md) — Create a new user in the Payward system.
   - `POST /b2b/users`
2. [Create Headless Subaccount](02_Create-Headless-Subaccount.md) — Create a lightweight account with only an external ID mapping, without user profile data.
   - `POST /b2b/subaccounts`
3. [Get Embed User](03_Get-Embed-User.md) — Retrieve a previously created user.
   - `GET /b2b/users/:user`
4. [Update Embed User](04_Update-Embed-User.md) — Update an existing user's profile.
   - `PATCH /b2b/users/:user`
