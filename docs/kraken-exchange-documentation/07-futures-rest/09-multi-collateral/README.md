# Multi Collateral

Leverage and PnL currency preference endpoints for multi-collateral futures contracts.

## Contents

1. [Get Leverage Setting](01_Get-Leverage-Setting.md) -- Return configured leverage preferences for multi-collateral futures.
   - `GET /derivatives/api/v3/leveragepreferences`
2. [Get PnL Currency Preference](02_Get-Pnl-Currency-Preference.md) -- Return the currency preference used for realizing PnL gains.
   - `GET /derivatives/api/v3/pnlpreferences`
3. [Set Leverage Setting](03_Set-Leverage-Setting.md) -- Set a contract's margin mode to isolated or cross margin with optional max leverage.
   - `PUT /derivatives/api/v3/leveragepreferences`
4. [Set PnL Currency Preference](04_Set-Pnl-Currency-Preference.md) -- Set the currency preference used for realizing PnL gains.
   - `PUT /derivatives/api/v3/pnlpreferences`
