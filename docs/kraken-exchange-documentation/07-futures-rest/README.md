# Futures REST API

REST API endpoints for Kraken Futures -- accounts, orders, market data, instruments, and transfers.

## Contents

1. [Account](01-account/) -- Wallet balances, open positions, portfolio margining parameters, and portfolio simulation.
2. [Assignment Program](02-assignment-program/) -- Manage futures contract assignment preferences and view assignment history.
3. [Charts](03-charts/) -- OHLC candle data, market analytics, liquidity pool statistics, and chart metadata.
4. [Fee Schedules](04-fee-schedules/) -- Fee schedule definitions and per-user trading volume tiers.
5. [General](05-general/) -- Trade fills, platform notifications, and historical funding rates.
6. [History](06-history/) -- Account and market history including execution, order, trigger, and price events.
7. [Instruments](07-instruments/) -- Instrument specifications, trading instruments, and market status details.
8. [Market Data](08-market-data/) -- Trade history, order book snapshots, and ticker data for listed contracts.
9. [Multi Collateral](09-multi-collateral/) -- Leverage and PnL currency preferences for multi-collateral futures.
10. [Order Management](10-order-management/) -- Placing, editing, cancelling, and querying orders including batch operations.
11. [RFQ](11-rfq/) -- Request for Quote workflow for block trade negotiation -- creating, offering, and accepting.
12. [Settings](12-settings/) -- Self-trade strategy, subaccount management, and trading capability controls.
13. [Transfers](13-transfers/) -- Fund transfers between accounts, subaccounts, and withdrawals to Spot.
