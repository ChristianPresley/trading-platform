# Portfolios

Embed API endpoints for managing and viewing user portfolio information, including historical balances and asset holdings.

## Contents

1. [Get Embed Portfolio History](01_Get-Embed-Portfolio-History.md) — Retrieve a portfolio's historical balances and valuations over time.
   - `GET /b2b/portfolio/:user/history`
2. [List Embed Portfolio Details](02_List-Embed-Portfolio-Details.md) — List owned assets in a user's portfolio.
   - `GET /b2b/portfolio/:user/details`
3. [Portfolios](03_Portfolios.md) — Overview of all portfolio endpoints including summaries, history, details, and transactions.
