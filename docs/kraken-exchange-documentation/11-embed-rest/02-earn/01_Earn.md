# Earn Overview

## Description

The Earn section of the Kraken Embed API provides endpoints for managing earn/staking functionality for users, including viewing earn summaries, listing available earn assets, and toggling auto-earn settings.

## Available Endpoints

### Get Earn Summary

Get the earn summary of a user, including current earn balances and rewards.

### List Earn Assets

List available earn assets and their current rates/terms.

### Toggle Auto-Earn

Toggle the auto-earn setting for a user, enabling or disabling automatic staking of eligible assets.

## Authentication

All earn endpoints require authenticated API credentials.

## Notes

- Earn functionality allows users to earn rewards on eligible crypto assets.
- Auto-earn automatically stakes eligible assets when enabled.
- Availability of earn features depends on the user's jurisdiction and account status.
- For individual endpoint details including parameters, response fields, and error codes, see the respective endpoint documentation pages.
