# Websocket V2

WebSocket API v2 (current) -- real-time market data, user data, and trading channels.

## Contents

1. [Admin](01-admin/) -- Connection lifecycle events: heartbeat, ping/pong, and system status.
2. [Market Data](02-market-data/) -- Public channels for L2/L3 order book, instrument reference data, OHLC, ticker, and trades.
3. [User Data](03-user-data/) -- Authenticated channels for balance streaming and order/trade execution events.
4. [User Trading](04-user-trading/) -- Authenticated methods for single/batch order placement, amendment, cancellation, and dead man's switch.
