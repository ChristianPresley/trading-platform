# Spot Example Clients

> Source: https://docs.kraken.com/api/docs/guides/spot-examples

## Example API Clients

Kraken recommends implementing their API with your own code while minimizing reliance on third-party solutions. The company is developing official Python and Golang clients that will be actively maintained. Until then, their Client Engagement team has compiled code snippets, examples and Postman collections available through their support portal.
