# Assignment Program

Futures contract assignment preference management -- add, delete, and query assignment programs.

## Contents

1. [Add Program](01_Add-Program.md) -- Add an assignment program preference for a futures contract.
   - `POST /assignmentprogram/add`
2. [Delete Program](02_Delete-Program.md) -- Delete an existing assignment program preference for a futures contract.
   - `POST /assignmentprogram/delete`
3. [Get Current](03_Get-Current.md) -- Return currently active assignment programs for the account.
   - `GET /assignmentprogram/current`
4. [Get History](04_Get-History.md) -- Return the history of assignment program preference changes.
   - `GET /assignmentprogram/history`
