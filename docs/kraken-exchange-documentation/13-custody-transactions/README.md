# Custody Transactions

Kraken Custody transaction endpoints for retrieving transaction details and history.

## Contents

1. [Get Custody Transaction](01_Get-Custody-Transaction.md) — Retrieve details for a specific custody transaction by its transaction ID.
   - `POST /0/private/GetCustodyTransaction`
2. [List Custody Transactions](02_List-Custody-Transactions.md) — Retrieve the transaction history for a specified vault.
   - `POST /0/private/ListCustodyTransactions`
