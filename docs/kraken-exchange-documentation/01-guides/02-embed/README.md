# Embed

Guides for integrating Kraken Embed (B2B/B2B2C): authentication, trading workflows, custom orders, and crypto funding.

## Contents

1. [Authentication](01_Authentication.md) — HMAC-SHA512 signing with API-Key, API-Sign, and API-Nonce headers, plus optional API versioning for Embed REST endpoints.
2. [Custom Orders](02_Custom-Orders.md) — Price-triggered order workflows (buy-the-dip, stop-loss, take-profit), prospective quotes, trigger conditions, and order lifecycle management.
3. [Deposits and Withdrawals](03_Deposits-And-Withdrawals.md) — Crypto deposit address creation, withdrawal workflows, method listing, and webhook-based status tracking via the Embed API.
4. [Your First Trade](04_Your-First-Trade.md) — Step-by-step quote-based trading workflow: requesting quotes, executing trades, and monitoring status with spend/receive quote types.
