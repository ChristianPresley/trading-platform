# Subaccounts

Endpoints for creating and managing subaccounts, including fund transfers between master and subaccounts.

## Contents

1. [Account Transfer](01_Account-Transfer.md) — Transfer funds between the master account and subaccounts.
   - `POST /0/private/AccountTransfer`
2. [Create Subaccount](02_Create-Subaccount.md) — Create a trading subaccount (must be called from the master account).
   - `POST /0/private/CreateSubaccount`
