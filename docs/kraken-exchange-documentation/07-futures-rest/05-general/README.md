# General

General-purpose endpoints -- trade fills, platform notifications, and historical funding rates.

## Contents

1. [Get Fills](01_Get-Fills.md) -- Return filled order information for all futures contracts on the authenticated account.
   - `GET /fills`
2. [Get Notifications](02_Get-Notifications.md) -- Return platform notifications such as maintenance windows and announcements.
   - `GET /notifications`
3. [Historical Funding Rates](03_Historical-Funding-Rates.md) -- Return historical funding rates for a given perpetual futures contract.
   - `GET /historical-funding-rates`
