# Assets

Embed API endpoints for querying available assets and exchange rates.

## Contents

1. [Get Embed Asset](01_Get-Embed-Asset.md) — Retrieve details for a specific asset.
   - `GET /b2b/assets/:asset`
2. [List Embed Asset Rates](02_List-Embed-Asset-Rates.md) — Get historical rates for a given asset, including timestamps and median daily prices.
   - `GET /b2b/assets/:asset/rates`
3. [List Embed Assets](03_List-Embed-Assets.md) — List all assets available on the platform with identifiers, types, and statuses.
   - `GET /b2b/assets`
