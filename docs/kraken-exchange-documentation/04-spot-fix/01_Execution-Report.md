# Execution Report

> Source: https://docs.kraken.com/api/docs/fix-api/er-fix

## Overview

As soon as Kraken Trading Engine accepts the order, all order responses will be sent using the Execution Report message. This includes any fills, order status changes, and confirmation of cancellation.

**FIX Message Type:** `8` (Execution Report)

## Notes

- For triggered orders, OrderType reflects the new execution model post-trigger.
- Fees are cumulative from previous trades.
- It can happen in some scenarios that trades are not reported in time through the FIX API. In that case the FIX API will detect it and send an ExecType=Restated execution report.
- Broker account orders always have NoAllocs=1; contact Account Manager for subaccount details.
