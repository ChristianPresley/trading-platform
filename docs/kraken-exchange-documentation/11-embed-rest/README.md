# Embed REST

Kraken Embed (white-label) REST API -- user management, quotes, assets, and verifications.

## Contents

1. [Assets](01-assets/) — Query available assets and exchange rates (3 endpoints).
2. [Earn](02-earn/) — Manage earn/staking functionality, summaries, and auto-earn settings.
3. [Portfolios](03-portfolios/) — View user portfolio balances, holdings, and history (2 endpoints + overview).
4. [Quotes](04-quotes/) — Request, retrieve, and execute trade quotes (6 endpoints).
5. [Users](05-users/) — Create, retrieve, and update user accounts (4 endpoints).
6. [Verifications](06-verifications/) — Submit identity verification documents for users (2 endpoints).
