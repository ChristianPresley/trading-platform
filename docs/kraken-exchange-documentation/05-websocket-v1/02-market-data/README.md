# Market Data

Real-time public market data channels for order book, candlestick, spread, ticker, and trade data.

## Contents

1. [Book](01_Book.md) -- Level 2 order book snapshots and incremental updates at configurable depth.
   - Channel: `book`
2. [OHLC](02_Ohlc.md) -- Open/High/Low/Close candlestick data for configurable interval periods.
   - Channel: `ohlc`
3. [Spread](03_Spread.md) -- Real-time best bid/ask spread data with volumes.
   - Channel: `spread`
4. [Ticker](04_Ticker.md) -- Level 1 ticker with real-time price, volume, and trade statistics.
   - Channel: `ticker`
5. [Trade](05_Trade.md) -- Real-time trade execution feed for currency pairs.
   - Channel: `trade`
