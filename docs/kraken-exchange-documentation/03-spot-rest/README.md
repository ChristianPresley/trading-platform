# Spot REST API

REST API endpoints for the Kraken spot exchange.

## Contents

1. [Account Data](01-account-data/) — Balances, orders, trades, ledgers, positions, and data exports (19 endpoints).
2. [Earn](02-earn/) — Allocate/deallocate funds to Earn strategies and check status (6 endpoints).
3. [Funding](03-funding/) — Deposits, withdrawals, wallet transfers, and associated status/methods (10 endpoints).
4. [Market Data](04-market-data/) — Asset info, order books, OHLC, trades, spreads, tickers, and system status (11 endpoints).
5. [Subaccounts](05-subaccounts/) — Create subaccounts and transfer funds between master/sub (2 endpoints).
6. [Trading](06-trading/) — Place, amend, edit, and cancel orders; batch operations; WebSocket tokens (9 endpoints).
7. [Transparency](07-transparency/) — MiFID II / MiCAR pre-trade and post-trade regulatory reporting (2 endpoints).
