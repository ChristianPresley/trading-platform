# Spot Exchange

Guides for Kraken Spot trading across FIX, REST, and WebSocket protocols, plus general trading concepts.

## Contents

1. [FIX](01-fix/) — FIX 4.4 protocol introduction, SenderCompID authentication, and order book checksum verification.
2. [General](02-general/) — Cross-protocol trading concepts: atomic amends, client order identifiers, error codes, Level 3 data, and rate limits.
3. [REST](03-rest/) — Spot REST API introduction, authentication, earn/staking, and call-counter rate limits.
4. [WebSockets](04-websockets/) — Spot WebSocket token authentication, book checksum calculation for v1/v2, and Level 3 checksum verification.
