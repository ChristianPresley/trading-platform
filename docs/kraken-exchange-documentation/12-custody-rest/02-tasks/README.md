# Tasks

Custody REST API endpoints for managing review tasks and their associated activities.

## Contents

1. [Get Custody Activity](01_Get-Custody-Activity.md) — Retrieve details for a specific task activity.
   - `POST /0/private/GetCustodyActivity`
2. [Get Custody Task](02_Get-Custody-Task.md) — Retrieve details for a specific task.
   - `POST /0/private/GetCustodyTask`
3. [List Custody Activities](03_List-Custody-Tasks-Activities.md) — Retrieve all activities matching specified filter criteria.
   - `POST /0/private/ListCustodyActivities`
4. [List Custody Tasks](04_List-Custody-Tasks.md) — Retrieve review tasks matching specified filter criteria.
   - `POST /0/private/ListCustodyTasks`
5. [Tasks](05_Tasks.md) — Overview of all task endpoints and related sections.
